# Quietware Mail · 清净邮件

AI 收件箱降噪:自动把邮件分流(重要/通知/营销/社交/其他)+ 一句话摘要 + 标记是否需回复。

## ⚠️ 诚实说明
当前是**用本地示例数据演示核心价值**,**没有接真实 IMAP/SMTP/邮箱 API**。
先把"AI 整理收件箱"的体验做出来验证价值,真实邮箱接入是后续工程(优先级中等,
见根 [调研文档](../../品类调研-去广告开源订阅产品.md))。

## 功能(✅ 可跑,基于示例数据)
- 原始收件箱视图(杂乱:重要/营销混在一起)
- 一键 AI 整理 → 按类别分组 + 摘要 + 待回复标记(复用 `@quietware/core` 的 `triageEmail`)

## 跑起来
```bash
pnpm --filter @quietware/mail dev   # web :5179  api :8793
```
需配 `ANTHROPIC_API_KEY` 才能看 AI 整理效果。

## 待办(让它变成真产品)
接 Gmail/Outlook API 或 IMAP、本地优先 + 加密、批量取消订阅营销邮件。
