# Quietware Mail · 清净邮件(🏗️ 脚手架)

> 状态:**立项占位**。

## 定位
干净、AI 整理(自动摘要/分类/降噪)的邮件客户端。参考 Spark / Superhuman。

## 评分(综合 3.30)
- 广告恶心度 3 · 付费意愿 4 · 可做性 3 · 低风险 4 · 迁移成本 2

## 差异化策略
卖点不是"去广告"(邮件广告没那么尖锐),而是 **AI 收件箱降噪**:自动把营销邮件、
通知、重要邮件分流,一句话摘要长邮件。这是 `@quietware/core` 的 `ai` 模块的天然延伸。

## 是否复用 core
**ai 模块复用**;feed 引擎不适用(改用 IMAP/SMTP)。
