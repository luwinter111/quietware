# Quietware Shop · 想买清单

干净的「想买清单」:手动记录想买的东西(标题/链接/心理价位/备注),本地持久化,不被推广打扰。

## ⚠️ 诚实说明
**不扒电商、不做竞价排名、不接导购佣金、不做自动比价**——那些要么踩反爬红线,
要么与"无广告/不打扰"的品牌定位冲突。这个品类在我们的定位下,能站得住的最小形态
就是一个干净、本地、不被打扰的清单(评分最低 2.45,见 [调研文档](../../品类调研-去广告开源订阅产品.md))。

## 功能(✅ 可跑)
- 增/删想买项,标记已购,本地持久化(`~/.quietware/shop.json`)

## 跑起来
```bash
pnpm --filter @quietware/shop dev   # web :5180  api :8794
```
