# Quietware Tools · 工具类(🏗️ 脚手架)

> 状态:**立项占位**。结构已建,等待实现。

## 定位
大陆免费工具类 App(输入法、清理、天气、相册)是广告与隐私重灾区:弹窗、开屏、偷数据。
卖点 = **无广告 + 不偷数据的信任品牌**。

## 评分(综合 4.25,候选第二)
- 广告恶心度 5 · 付费意愿 4 · 可做性 4 · 低风险 5 · 迁移成本 2

## 差异化策略
先做**轻量工具**(剪贴板历史、截图标注、文件清理)建立"隐私+无广告"品牌,
再切更重的输入法(词库/引擎工程量大、切换习惯成本高,是最大障碍)。

## 是否复用 core
**否**。工具类逻辑自包含,不依赖 feed 引擎。但可复用 `@quietware/core` 的 `ai` 模块
(如剪贴板内容的 AI 整理、截图 OCR 摘要)。
